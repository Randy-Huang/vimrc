MANUALLY INSTALL
----------------

1. Check out from github

        git clone git://github.com/Randy-Huang/vimrc.git ~/.vim

2. Install ~/.vimrc        
    
        cd ~/.vim
        ./auto-install.sh

3. Complish all!
